    Este documento se basa en "Metagenomics workshop" Claudia Zirión Martínez; Diego Garfias Gallegos; Tania Vanessa Arellano Fernández; Aarón Espinosa Jaime; Edder D Bustos Díaz; José Abel Lovaco Flores; Luis Gerardo Tejero Gómez; J Abraham Avelar Rivas; Nelly Sélem (March , 2023) A Data Carpentry- Style Metagenomics Workshop.

    • Fechas: 22 y 23 de Mayo 2025

    • Duración del curso: 14 horas

    Instructorxs:

    • Aaron Espinosa Jaime - MSc, UGA, CINVESTAV Irapuato.
    • Yahir Cabrera González - IBT, IBC, UNACH.

    Resumen:

    Este tutorial está diseñado para enseñar a los investigadorxs, conceptos, habilidades y herramientas básicas para trabajar con datos, permitiéndoles optimizar su trabajo con mayor rapidez y eficiencia.

    Objetivos:

    Este taller utiliza el enfoque de Data Carpentry para enseñar:

    1. Buenas prácticas para la organización de proyectos y datos bioinformáticos.

    2. Uso de utilidades de línea de comandos para la gestión y análisis de datos.

    3. Herramientas de línea de comandos para analizar la calidad de secuencias.

    4. Uso de R Studio y bibliotecas de R para comparar la diversidad entre muestras.

    5. Conexión y uso de servidores remotos.

    Citar y reutilizar el material del curso

    Los datos del curso se pueden reutilizar y adaptar libremente con la debida atribución. Todos los datos de los cursos de estos repositorios están sujetos a la licencia Attribution-NonCommercial-ShareAlike 4.0 International (CC BY-NC-SA 4.0).
